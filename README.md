# EternityCorrect
releasesの欄からaptファイルをダウンロードしてAndroid上で使ってください。

このアプリは、立方体上に配置されているT字もしくはI字のパーツを回転させることで、パーツを立方体上で一周つなげればクリアとなるパズルゲームです。
